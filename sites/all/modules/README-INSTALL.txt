Note for Geofield Module installation procedure
-----------------------------------------------
Geofield requires GeoPHP module which provides integration with the geoPHP library. On production please follow detailed instruction how to setup GEOS PHP extension on the server for performance improvment.  https://github.com/phayes/geoPHP/wiki/GEOS
